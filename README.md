## ğŸ“¦Installation

Install and set up Nowcast locally:

### Clone the repository
git clone https://github.com/AyanAlikhan11/Nowcast.git

### Open project folder
cd Nowcast

### Install dependencies
npm install

### Create a file named .env.local:
